# AKB Football Academy

A responsive multi-page website for AKB Football Academy, showcasing training programs, coaches, facilities, and registration for players of all ages.

## Pages
- Home
- About
- Programs
- Coaches
- Gallery
- Testimonials
- Contact

## Technologies
- HTML5
- CSS3
- JavaScript (ES6+)
- Bootstrap 5

## Features
- Responsive design (mobile, tablet, desktop)
- Filterable gallery by training category
- Contact/registration form with validation
